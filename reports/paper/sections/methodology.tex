\section{Methodology}

This study follows a machine learning pipeline composed of data preprocessing,
feature engineering, model training, and evaluation.

\subsection{Dataset}

The dataset used in this study contains clinical attributes related to kidney
disease diagnosis.

\subsection{Preprocessing}

Missing values were handled, categorical variables were encoded, and numerical
features were normalized.

\subsection{Model Training}

Several machine learning models were trained and evaluated using cross-validation.
