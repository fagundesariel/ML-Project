\begin{abstract}
This paper presents a machine learning approach for predicting chronic kidney
disease using clinical data. The proposed methodology includes data 
preprocessing, model training, and evaluation of predictive performance.
\end{abstract}
     
\begin{resumo}
Este artigo apresenta uma abordagem de aprendizado de máquina para predição 
de doença renal crônica utilizando dados clínicos. A metodologia proposta 
inclui pré-processamento de dados, treinamento de modelos e avaliação de
desempenho preditivo.
\end{resumo}
